# gorazdo-plugins — Claude Code Marketplace

A plugin marketplace for Claude Code.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `untangle-with-ralph-loop` | Iterative cognitive complexity reducer and test coverage generator for TypeScript/JavaScript |

## Usage

```bash
# Add marketplace (one-time)
claude plugin marketplace add gorazdo/claude-marketplace

# Install a plugin
claude plugin install untangle-with-ralph-loop@gorazdo-plugins
```
